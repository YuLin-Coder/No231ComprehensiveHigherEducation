## 项目介绍
基于springboot的高校教育综合管理系统：前端 html、jquery、layui，后端 maven、springmvc、spring、mybatis；角色分为管理员、老师、学生；集成宿舍管理、教评管理、排课管理、考试管理等功能于一体的系统。
